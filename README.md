1. What is the difference between var, let, and const?
ans:  The difference between var, let, and const are var is function scope on the other hand let and const are block scope. Variables declared with var are hoisted but variables declared with let or const are not hoisted.

2. What is the spread operator (...)?
ans: The spread operator (...) is a powerful tool introduced in ES6 that allows you to "expand" or unpack an iterable—like an array, string, or object—into individual elements.

3. What is the difference between map(), filter(), and forEach()?
ans: The difference between map(), filter(), and forEach() are map() transforms each element create a new array. filter() select select specific items and create a shorter or empty new array. forEach() just run a function for each item.
